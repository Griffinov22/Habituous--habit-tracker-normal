# Centering a div with position:absolute

Make sure you have a width and height

Horizontally
element {
position: absolute;
left: 0;
right: 0;
margin: 0 auto;
}

Vertically
element {
position: absolute;
top: 0;
bottom: 0;
margin: auto 0;
}
